# Realtime web

In this repo, you can find my work for the course `realtime web` for my minor `everything-web` at the HvA Amsterdam.

## Installation

```
$ git clone https://github.com/Frankwarnaar/minor-realtime-web.git
$ npm install
$ gulp build
$ npm start
```
